CONFIGURAÇÃO DO DOCUSAURUS

1. Criação dos endpoints
	1.1 Copiar o arquivo json do Swagger para o projeto do docusaurus;
	1.2 Inserir arquivo com IP's antes de "paths".
		"servers": [
			{
			"url": "http://localhost:9010"
			}
		],  
	1.3 Configurar caminho do arquivo json na configuração do plugin da API;
	1.4 Configurar a pasta onde serão gerados os arquivos MDX;
	1.5 Comentar a linha sidebarPath: 'docs/api-v1/sidebar.ts' no "docusaurus.config.ts";
	1.6 Executar o comando: yarn docusaurus gen-api-docs all;
	1.7 No arquivo "sidebar" gerado, alterar a última linha 
		De: "export default sidebar.apisidebar";
		Para: "export default sidebar;
	1.8 Descomentar a linha sidebarPath: 'docs/api-v1/sidebar.ts' no "docusaurus.config.ts";
